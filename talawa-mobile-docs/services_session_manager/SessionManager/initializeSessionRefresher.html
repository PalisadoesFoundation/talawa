initializeSessionRefresher method

Timer initializeSessionRefresher()

Initializes as session refresher.

Invokes refreshSession periodically at regular refresh intervals.

params: None

returns:

  • Timer: refresh timer.

Implementation

Timer initializeSessionRefresher() {
  return Timer.periodic(
    const Duration(seconds: _refreshInterval),
    (Timer timer) async {
      refreshSession();
    },
  );
}
