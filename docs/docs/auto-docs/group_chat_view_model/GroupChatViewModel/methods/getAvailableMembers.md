# Method: `getAvailableMembers`

## Description

Gets available members for adding to a group chat.

 Returns organization members who are not already in the chat.

 **params**:
 * `chatId`: The ID of the group chat

 **returns**:
 * `List&lt;User&gt;`: List of available organization members

## Return Type
`List&lt;User&gt;`

## Parameters

- `chatId`: `String`
