# Method: `sendMessageToGroupChat`

## Description

Send text message to a group chat.

 **params**:
 * `chatId`: id of a group chat to which message is to be sent.
 * `messageText`: text of a message to be sent.

 **returns**:
   None

## Return Type
`Future&lt;void&gt;`

## Parameters

- `chatId`: `String`
- `messageText`: `String`
