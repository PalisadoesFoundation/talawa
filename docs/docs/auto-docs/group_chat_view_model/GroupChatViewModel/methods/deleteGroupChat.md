# Method: `deleteGroupChat`

## Description

Deletes a group chat (admin only).

 **params**:
 * `chatId`: The ID of the group chat to delete

 **returns**:
 * `Future&lt;bool&gt;`: True if group was deleted successfully, false otherwise

## Return Type
`Future&lt;bool&gt;`

## Parameters

- `chatId`: `String`
