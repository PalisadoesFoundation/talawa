# Method: `pushNamed`

## Description

Push a named route onto the navigator.

 

 

 

 

 Typical usage is as follows:

 ```dart
 void  
 ```
 

 See also:

  * [restorablePushNamed], which pushes a route that can be restored
    during state restoration.

## Return Type
`Future<T?>`

- `routeName`: `String`
