# Method: `push`

## Description

Push the given route onto the navigator.

 

 

 

 Typical usage is as follows:

 ```dart
 void  
 ```
 

 See also:

  * [restorablePush], which pushes a route that can be restored during
    state restoration.

## Return Type
`Future<T?>`

## Parameters

- `route`: `Route&lt;T&gt;`
