# Method: `formatMediumDate`

## Description

Formats the date using a medium-width format.

 Abbreviates month and days of week. This appears in the header of the date
 picker invoked using [showDatePicker].

 Examples:

 - US English: Wed, Sep 27
 - Russian: ср, сент. 27

## Return Type
`String`

## Parameters

- `date`: `DateTime`
