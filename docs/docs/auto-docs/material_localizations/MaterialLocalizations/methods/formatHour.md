# Method: `formatHour`

## Description

Formats [TimeOfDay.hour] in the given time of day according to the value
 of [timeOfDayFormat].

 If [alwaysUse24HourFormat] is true, formats hour using [HourFormat.HH]
 rather than the default for the current locale.

## Return Type
`String`

## Parameters

- `timeOfDay`: `TimeOfDay`
- ``: `dynamic`
