# Method: `forTap`

## Description

Provides platform-specific feedback for a tap.

 On Android the click system sound is played. On iOS this is a no-op.

 See also:

  * [wrapForTap] to trigger platform-specific feedback before executing a
    [GestureTapCallback].

## Return Type
`Future&lt;void&gt;`

## Parameters

- `context`: `BuildContext`
