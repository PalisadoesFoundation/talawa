# Method: `validatePasswordConfirm`

## Description

Method to valid password re-entered for confirmation.

 **params**:
 * `value`: the entered password
 * `comparator`: the original password

 **returns**:
 * `String?`: error message if password is invalid.

## Return Type
`String?`

## Parameters

- `value`: `String`
- `comparator`: `String`
