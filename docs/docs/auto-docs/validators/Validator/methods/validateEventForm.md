# Method: `validateEventForm`

## Description

Method to validate event form.

 **params**:
 * `value`: the value of the field
 * `label`: the (optional) label of the field

 **returns**:
 * `String?`: error message if field is invalid.

## Return Type
`String?`

## Parameters

- `value`: `String`
- `label`: `String?`
