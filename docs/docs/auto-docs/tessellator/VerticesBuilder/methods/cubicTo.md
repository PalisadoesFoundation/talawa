# Method: `cubicTo`

## Description

Adds a cubic Bezier curve with x1,y1 as the first control point, x2,y2 as
 the second control point, and end point x3,y3.

## Return Type
`void`

## Parameters

- `x1`: `double`
- `y1`: `double`
- `x2`: `double`
- `y2`: `double`
- `x3`: `double`
- `y3`: `double`
