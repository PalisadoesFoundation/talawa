<div class="section desc markdown">

The location in the tree where this widget builds.

The framework associates `State` objects with a `BuildContext` after
creating them with `StatefulWidget.createState` and before calling
[initState](../../views_after_auth_screens_events_create_agenda_item_page/CreateAgendaItemPageState/initState.md).
The association is permanent: the `State` object will never change its
`BuildContext`. However, the `BuildContext` itself can be moved around
the tree.

After calling
[dispose](../../views_after_auth_screens_events_create_agenda_item_page/CreateAgendaItemPageState/dispose.md),
the framework severs the `State` object's connection with the
`BuildContext`.
