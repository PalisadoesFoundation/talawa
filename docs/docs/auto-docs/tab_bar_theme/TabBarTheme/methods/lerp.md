# Method: `lerp`

## Description

Linearly interpolate between two tab bar themes.

 

 This method is obsolete and will be deprecated in a future release:
 please use the [TabBarThemeData.lerp] instead.

## Return Type
`TabBarTheme`

## Parameters

- `a`: `TabBarTheme`
- `b`: `TabBarTheme`
- `t`: `double`
