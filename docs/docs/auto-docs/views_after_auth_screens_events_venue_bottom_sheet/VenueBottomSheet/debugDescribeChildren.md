<div class="section desc markdown">

Returns a list of `DiagnosticsNode` objects describing this node's
children.

Children that are offstage should be added with `style` set to
`DiagnosticsTreeStyle.offstage` to indicate that they are offstage.

The list must not contain any null entries. If there are explicit null
children to report, consider `DiagnosticsNode.message` or
`DiagnosticsProperty<Object>` as possible `DiagnosticsNode` objects to
provide.

Used by
[toStringDeep](../../views_after_auth_screens_events_venue_bottom_sheet/VenueBottomSheet/toStringDeep.md),
[toDiagnosticsNode](../../views_after_auth_screens_events_venue_bottom_sheet/VenueBottomSheet/toDiagnosticsNode.md)
and
[toStringShallow](../../views_after_auth_screens_events_venue_bottom_sheet/VenueBottomSheet/toStringShallow.md).

See also:

- `RenderTable.debugDescribeChildren`, which provides high quality
  custom descriptions for its child nodes.
