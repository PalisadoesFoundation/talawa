# Method: `getImageFormatInfo`

## Description

Detects the image format and extension from file path.

 **params**:
 * `filePath`: Path to the image file

 **returns**:
 * `Map&lt;String, dynamic&gt;`: Map containing 'format' (CompressFormat) and 'extension' (String)

## Return Type
`Map&lt;String, dynamic&gt;`

## Parameters

- `filePath`: `String`
