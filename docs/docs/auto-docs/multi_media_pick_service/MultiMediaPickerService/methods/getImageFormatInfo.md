# Method: `getImageFormatInfo`

## Description

Detects the image format and extension from file path.

 **params**:
 * `filePath`: Path to the image file

 **returns**:
 * `Map<String, dynamic>`: Map containing 'format' (CompressFormat) and 'extension' (String)

## Return Type
`Map<String, dynamic>`

## Parameters

- `filePath`: `String`
