# Method: `getOrgMembersList`

## Description

Retrieves a list of organization members.

 **params**:
 * `orgId`: The ID of the organization to fetch members from.

 **returns**:
 * `Future&lt;List&lt;User&gt;&gt;`: A promise that will be fulfilled
 with the list of organization members.

## Return Type
`Future&lt;List&lt;User&gt;&gt;`

## Parameters

- `orgId`: `String`
