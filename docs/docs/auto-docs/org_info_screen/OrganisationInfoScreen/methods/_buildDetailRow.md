# Method: `_buildDetailRow`

## Description

Builds a detail row with a label and value.

 **params**:
 * `label`: The label for the detail.
 * `value`: The value for the detail.

 **returns**:
 * `Widget`:  A widget containing the label and value in a row.

## Return Type
`Widget`

## Parameters

- `label`: `String`
- `value`: `String?`
