# Method: `lerp`

## Description

Linearly interpolate between two dialog themes.

 

 This method is obsolete and will be deprecated in a future release:
 please use the [DialogThemeData.lerp] instead.

## Return Type
`DialogTheme`

## Parameters

- `a`: `DialogTheme?`
- `b`: `DialogTheme?`
- `t`: `double`
