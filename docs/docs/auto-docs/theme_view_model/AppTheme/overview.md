# Overview for `AppTheme`

## Description

AppTheme class is a type of ViewModel to serve data from model to views in the context of App Themes.

 Methods include:
 * `switchTheme`

## Dependencies

- BaseModel

## Members

- **key**: `String`
  Key for dynamic theme.

- **_pref**: `SharedPreferences`
- **_isDarkMode**: `bool`
