# Method: `validateMemberRemoval`

## Description

Validates if a member can be removed from a group chat.

 **params**:
 * `chat`: The chat to validate
 * `memberId`: The ID of the member to remove
 * `currentUserId`: The ID of the current user

 **returns**:
 * `Map&lt;String, dynamic&gt;`: Map with 'isValid' boolean and 'error' string if invalid
