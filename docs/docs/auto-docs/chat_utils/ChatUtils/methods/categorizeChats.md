# Method: `categorizeChats`

## Description

Categorizes chats into groups and direct chats.

 **params**:
 * `chats`: List of all chats to categorize

 **returns**:
 * `Map&lt;String, List&lt;Chat&gt;&gt;`: Map with 'groups' and 'direct' keys containing categorized chats

## Return Type
`Map&lt;String, List&lt;Chat&gt;&gt;`

## Parameters

- `chats`: `List&lt;Chat&gt;`
