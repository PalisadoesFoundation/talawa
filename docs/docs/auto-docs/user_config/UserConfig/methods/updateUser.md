# Method: `updateUser`

## Description

Updates the user details.

 **params**:
 * `updatedUserDetails`: `User` type variable containing
  all the details of an user need to be updated.

 **returns**:
 * `Future&lt;bool&gt;`: returns future of bool type.

## Return Type
`Future&lt;bool&gt;`

## Parameters

- `updatedUserDetails`: `User`
