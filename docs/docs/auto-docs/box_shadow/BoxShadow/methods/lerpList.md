# Method: `lerpList`

## Description

Linearly interpolate between two lists of box shadows.

 If the lists differ in length, excess items are lerped with null.

 

## Return Type
`List&lt;BoxShadow&gt;?`

## Parameters

- `a`: `List&lt;BoxShadow&gt;?`
- `b`: `List&lt;BoxShadow&gt;?`
- `t`: `double`
