# Method: `registerUser`

## Description

Mutation to register a user.


 **params**:
 * `name`: user's data.
 * `email`: user's data.
 * `password`: user's data.
 * `selectedOrganization`: ID of the selected organization.

 **returns**:
 * `String`: Return the mutation in string type to be passed to graphql client.

- `email`: `String`
- `password`: `String`
- `selectedOrganization`: `String?`
