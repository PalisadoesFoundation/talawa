# Overview for `DefaultCupertinoLocalizations`

## Description

US English strings for the Cupertino widgets.

## Dependencies

- CupertinoLocalizations

## Members

- **_shortWeekdays**: `List&lt;String&gt;`
  Short version of days of week.

- **_shortMonths**: `List&lt;String&gt;`
- **_months**: `List&lt;String&gt;`
- **delegate**: `LocalizationsDelegate&lt;CupertinoLocalizations&gt;`
  A [LocalizationsDelegate] that uses [DefaultCupertinoLocalizations.load]
 to create an instance of this class.

## Constructors

### Unnamed Constructor
Constructs an object that defines the cupertino widgets' localized strings
 for US English (only).

 [LocalizationsDelegate] implementations typically call the static [load]
 function, rather than constructing this class directly.
