# Method: `containsAll`

## Description

Whether this set contains all the elements of [other].
 ```dart
 final characters = <String>;
 final containsAB = characters.;
 print(containsAB); // true
 final containsAD = characters.;
 print(containsAD); // false
 ```

## Return Type
`bool`

## Parameters

- `other`: `Iterable<Object?>`
