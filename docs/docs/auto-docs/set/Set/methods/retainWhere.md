# Method: `retainWhere`

## Description

Removes all elements of this set that fail to satisfy [test].
 ```dart
 final characters = <String>;
 characters.retainWhere(
     (element) => element.startsWith('B') || element.startsWith('C'));
 print(characters); // 
 ```

## Return Type
`void`
