# Method: `addAll`

## Description

Adds all [elements] to this set.

 Equivalent to adding each element in [elements] using [add],
 but some collections may be able to optimize it.
 ```dart
 final characters = <String>;
 characters.;
 print(characters); // 
 ```

## Return Type
`void`

## Parameters

- `elements`: `Iterable&lt;E&gt;`
