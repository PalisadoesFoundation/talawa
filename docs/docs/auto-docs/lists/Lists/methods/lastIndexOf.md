# Method: `lastIndexOf`

## Description

/**
   * Returns the last index in the array [a] of the given [element], starting
   * the search at index [startIndex] to 0.
   * Returns -1 if [element] is not found.
   */

## Return Type
`int`

## Parameters

- `a`: `List`
- `element`: `Object`
- `startIndex`: `int`
