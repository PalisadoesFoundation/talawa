# Method: `getRange`

## Description

/**
   * Returns a sub list copy of this list, from [start] to
   * [end] ([end] not inclusive).
   * Returns an empty list if [length] is 0.
   * It is an error if indices are not valid for the list, or
   * if [end] is before [start].
   */

## Return Type
`List`

## Parameters

- `a`: `List`
- `start`: `int`
- `end`: `int`
- `accumulator`: `List`
