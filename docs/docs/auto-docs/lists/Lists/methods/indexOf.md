# Method: `indexOf`

## Description

/**
   * Returns the index in the array [a] of the given [element], starting
   * the search at index [startIndex] to [endIndex] (exclusive).
   * Returns -1 if [element] is not found.
   */

## Return Type
`int`

## Parameters

- `a`: `List`
- `element`: `Object`
- `startIndex`: `int`
- `endIndex`: `int`
