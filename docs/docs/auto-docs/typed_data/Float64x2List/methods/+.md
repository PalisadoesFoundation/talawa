# Method: `+`

## Description

The concatenation of this list and [other].

 If [other] is also a [Float64x2List], the result is a new [Float64x2List],
 otherwise the result is a normal growable `List&lt;Float64x2&gt;`.

## Return Type
`List&lt;Float64x2&gt;`

## Parameters

- `other`: `List&lt;Float64x2&gt;`
