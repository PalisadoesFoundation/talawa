# Method: `+`

## Description

The concatenation of this list and [other].

 If [other] is also a [Float32x4List], the result is a new [Float32x4List],
 otherwise the result is a normal growable `List&lt;Float32x4&gt;`.

## Return Type
`List&lt;Float32x4&gt;`

## Parameters

- `other`: `List&lt;Float32x4&gt;`
