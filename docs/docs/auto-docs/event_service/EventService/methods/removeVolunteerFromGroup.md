# Method: `removeVolunteerFromGroup`

## Description

This function is used to remove a volunteer from a group.

 **params**:
 * `variables`: this will be `map` type and contain the ID of the volunteer to be removed.

 **returns**:
 * `Future&lt;dynamic&gt;`: Information about the removed volunteer.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `variables`: `Map&lt;String, dynamic&gt;`
