# Method: `fetchAttendeesByEvent`

## Description

This function is used to fetch all registrants of an event.

 **params**:
 * `eventId`: id of an event.

 **returns**:
 * `Future&lt;dynamic&gt;`: Information about event registrants.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `eventId`: `String`
