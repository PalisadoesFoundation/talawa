# Method: `fetchVolunteerGroupsByEvent`

## Description

This function is used to fetch all volunteer groups for an event.

 **params**:
 * `eventId`: Id of the event to fetch volunteer groups.

 **returns**:
 * `Future&lt;List&lt;EventVolunteerGroup&gt;&gt;`: returns the list of volunteer groups

## Return Type
`Future&lt;List&lt;EventVolunteerGroup&gt;&gt;`

## Parameters

- `eventId`: `String`
