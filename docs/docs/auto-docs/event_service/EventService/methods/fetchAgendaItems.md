# Method: `fetchAgendaItems`

## Description

This function is used to fetch all agenda items for a specific organization.

 **params**:
 * `eventId`: ID of the event to fetch agenda items.

 **returns**:
 * `Future&lt;dynamic&gt;`: A list of agenda items for the specified organization.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `eventId`: `String`
