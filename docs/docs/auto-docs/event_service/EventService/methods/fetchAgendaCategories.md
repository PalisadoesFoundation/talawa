# Method: `fetchAgendaCategories`

## Description

This function is used to create an agenda item.

 **params**:
 * `orgId`: ID of organisation to fetch categories.

 **returns**:
 * `Future&lt;dynamic&gt;`: Information about the created agenda item.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `orgId`: `String`
