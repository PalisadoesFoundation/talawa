# Method: `removeVolunteerGroup`

## Description

This function is used to remove a volunteer group.

 **params**:
 * `variables`: This will be a `map` type and contain the ID of the volunteer group to be deleted.

 **returns**:
 * `Future&lt;dynamic&gt;`: Information about the removed volunteer group.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `variables`: `Map&lt;String, dynamic&gt;`
