# Method: `createVolunteerGroup`

## Description

This function is used to create a volunteer group.

 **params**:
 * `variables`: this will be `map` type and contain all the volunteer group details need to be created.

 **returns**:
 * `Future&lt;dynamic&gt;`: Information about the created volunteer group.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `variables`: `Map&lt;String, dynamic&gt;`
