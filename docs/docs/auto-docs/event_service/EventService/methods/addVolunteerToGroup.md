# Method: `addVolunteerToGroup`

## Description

This function is used to add a volunteer to a group.

 **params**:
 * `variables`: this will be `map` type and contain all the details needed to add a volunteer to a group.

 **returns**:
 * `Future&lt;dynamic&gt;`: Information about the added volunteer.

## Return Type
`Future&lt;dynamic&gt;`

## Parameters

- `variables`: `Map&lt;String, dynamic&gt;`
