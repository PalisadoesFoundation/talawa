# Overview for `RenderSliverAnimatedOpacity`

## Description

Makes its sliver child partially transparent, driven from an [Animation].

 This is a variant of [RenderSliverOpacity] that uses an [Animation&lt;double&gt;]
 rather than a [double] to control the opacity.

## Dependencies

- RenderProxySliver, RenderAnimatedOpacityMixin

## Constructors

### Unnamed Constructor
Creates a partially transparent render object.
