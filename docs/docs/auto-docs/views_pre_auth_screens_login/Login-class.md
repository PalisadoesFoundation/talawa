# <span class="kind-class">Login</span> class

</div>

<div class="section desc markdown">

This is the login widget.

There are two input fiels. The first one takes in the email and the
second one takes in the password of the user. There is also a "Forgot
Password" text button, which directs to the "recover password" screen
when pressed. There is a login button to initiate the login. At the
bottom there is option to login directly with previous email.(This
option is only available if user select remember me while logging out)

</div>
