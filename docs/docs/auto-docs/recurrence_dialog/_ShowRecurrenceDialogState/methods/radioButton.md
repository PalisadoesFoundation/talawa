# Method: `radioButton`

## Description

Custom radio list tile for recurrence options.

 **params**:
 * `frequency`: Frequency of the event (DAILY, WEEKLY, MONTHLY, YEARLY)
 * `interval`: Interval between recurrences
 * `weekDays`: List of week days for weekly recurrence

 **returns**:
 * `RadioListTile&lt;String&gt;`: Radio list tile widget

## Return Type
`RadioListTile&lt;String&gt;`

## Parameters

- `frequency`: `String`
- `interval`: `int`
- `weekDays`: `List&lt;String&gt;?`
