# Overview for `PaginatedDataTableState`

## Description

Holds the state of a [PaginatedDataTable].

 The table can be programmatically paged using the [pageTo] method.

## Dependencies

- State

## Members

- **_firstRowIndex**: `int`
- **_rowCount**: `int`
- **_rowCountApproximate**: `bool`
- **_selectedRowCount**: `int`
- **_rows**: `Map&lt;int, DataRow?&gt;`
- **_tableKey**: `GlobalKey`
