<div class="section desc markdown">

Called when a dependency of this `State` object changes.

For example, if the previous call to
[build](../../views_after_auth_screens_events_create_event_page/CreateEventPageState/build.md)
referenced an `InheritedWidget` that later changed, the framework would
call this method to notify this object about the change.

This method is also called immediately after
[initState](../../views_after_auth_screens_events_create_event_page/CreateEventPageState/initState.md).
It is safe to call `BuildContext.dependOnInheritedWidgetOfExactType`
from this method.

Subclasses rarely override this method because the framework always
calls
[build](../../views_after_auth_screens_events_create_event_page/CreateEventPageState/build.md)
after a dependency changes. Some subclasses do override this method
because they need to do some expensive work (e.g., network fetches) when
their dependencies change, and that work would be too expensive to do
for every build.
