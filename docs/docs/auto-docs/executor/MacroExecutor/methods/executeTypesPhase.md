# Method: `executeTypesPhase`

## Description

Runs the type phase for [macro] on a given [declaration].

 Throws an exception if there is an error executing the macro.

## Return Type
`Future&lt;MacroExecutionResult&gt;`

## Parameters

- `macro`: `MacroInstanceIdentifier`
- `target`: `MacroTarget`
- `introspector`: `TypePhaseIntrospector`
