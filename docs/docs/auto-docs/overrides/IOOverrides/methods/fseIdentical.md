# Method: `fseIdentical`

## Description

Asynchronously returns `true` if [path1] and [path2] are paths to the
 same file system object.

 When this override is installed, this function overrides the behavior of
 `FileSystemEntity.identical`.

## Return Type
`Future&lt;bool&gt;`

## Parameters

- `path1`: `String`
- `path2`: `String`
