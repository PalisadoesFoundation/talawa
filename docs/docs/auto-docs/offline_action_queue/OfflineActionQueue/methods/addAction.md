# Method: `addAction`

## Description

Adds an action to the queue with a TTL.

 **params**:
 * `action`: the action to be added.

 **returns**:
 * `Future&lt;bool&gt;`: returns true if the action was added successfully, otherwise false.

## Return Type
`Future&lt;bool&gt;`

## Parameters

- `action`: `CachedUserAction`
