# Method: `subscribeToChatMessages`

## Description

Subscribes to real-time chat messages for a specific chat.

 **params**:
 * `chatId`: The ID of the chat to subscribe to

 **returns**:
 * `Stream&lt;ChatMessage&gt;`: Stream of incoming messages for the specified chat

## Return Type
`Stream&lt;ChatMessage&gt;`

## Parameters

- `chatId`: `String`
