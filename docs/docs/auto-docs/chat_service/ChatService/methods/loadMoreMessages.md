# Method: `loadMoreMessages`

## Description

Loads more messages (older messages) for a specific chat.

 **params**:
 * `chatId`: The ID of the chat

 **returns**:
 * `Future&lt;List&lt;ChatMessage&gt;&gt;`: List of older messages or empty list if failed

## Return Type
`Future&lt;List&lt;ChatMessage&gt;&gt;`

## Parameters

- `chatId`: `String`
