# Method: `nextDouble`

## Description

Generates a non-negative random floating point value uniformly distributed
 in the range from 0.0, inclusive, to 1.0, exclusive.

 Example:
 ```dart
 var doubleValue = .; // Value is >= 0.0 and < 1.0.
 doubleValue = . * 256; // Value is >= 0.0 and < 256.0.
 ```

## Return Type
`double`
