# Overview for `TlsProtocolVersion`

## Description

A Transport Layer Security (TLS) version.

 Only TLS versions supported by `dart:io` are included.

## Members

- **tls1_2**: `dynamic`
  Transport Layer Security (TLS) Protocol Version 1.2.
