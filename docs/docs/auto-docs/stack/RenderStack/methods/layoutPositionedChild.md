# Method: `layoutPositionedChild`

## Description

Lays out the positioned `child` according to `alignment` within a Stack of `size`.

 Returns true when the child has visual overflow.

## Return Type
`bool`

## Parameters

- `child`: `RenderBox`
- `childParentData`: `StackParentData`
- `size`: `Size`
- `alignment`: `Alignment`
