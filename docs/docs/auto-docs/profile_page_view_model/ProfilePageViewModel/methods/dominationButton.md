# Method: `dominationButton`

## Description

This widget returns button for domination.

 **params**:
 * `amount`: donation Amount.
 * `context`: BuildContext.
 * `setter`: `Function` type, which on tap set the amount to `donationAmount`.

 **returns**:
 * `Widget`: Icon Button

## Return Type
`Widget`

## Parameters

- `amount`: `String`
- `context`: `BuildContext`
- `setter`: `void Function(void )`
