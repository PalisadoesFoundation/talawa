# Method: `iconButton`

## Description

This widget returns the button for social media sharing option.

 **params**:
 * `icon`: This is Widget type with icon details.
 * `onTap`: This is Function which invoke on tap.

 **returns**:
 * `Widget`: Icon Button

## Return Type
`Widget`

## Parameters

- `icon`: `Widget`
- `onTap`: `void `
