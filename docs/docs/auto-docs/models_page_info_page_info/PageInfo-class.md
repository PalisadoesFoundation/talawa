# <span class="kind-class">PageInfo</span> class

</div>

<div class="section desc markdown">

This is class is used to represent pagination information for a list of
items.

variables: startCursor: start cursor for pagination endCursor: end
cursor for pagination hasNextPage: boolean indicating if there are more
items after the current page hasPreviousPage: boolean indicating if
there are items before the current page
