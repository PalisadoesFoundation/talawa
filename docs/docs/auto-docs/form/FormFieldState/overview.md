# Overview for `FormFieldState`

## Description

The current state of a [FormField]. Passed to the [FormFieldBuilder] method
 for use in constructing the form field's widget.

## Dependencies

- State, RestorationMixin

## Members

- **_value**: `T?`
- **_errorText**: `RestorableStringN`
- **_hasInteractedByUser**: `RestorableBool`
- **_focusNode**: `FocusNode`
