# Overview for `RenderAnimatedOpacity`

## Description

Makes its child partially transparent, driven from an [Animation].

 This is a variant of [RenderOpacity] that uses an [Animation&lt;double&gt;] rather
 than a [double] to control the opacity.

## Dependencies

- RenderProxyBox, RenderAnimatedOpacityMixin

## Constructors

### Unnamed Constructor
Creates a partially transparent render object.
