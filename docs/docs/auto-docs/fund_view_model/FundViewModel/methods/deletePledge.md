# Method: `deletePledge`

## Description

This function deletes a pledge.

 **params**:
 * `pledgeId`: id of the pledge to be deleted
 * `campaignId`: id of the campaign the pledge belongs to

 **returns**:
   None

## Return Type
`Future&lt;void&gt;`

## Parameters

- `pledgeId`: `String`
- `campaignId`: `String`
