# Method: `getEventsForCalendar`

## Description

Fetches events for the calendar within a specified date range.

 **params**:
 * `start`: DateTime representing the start date of the range.
 * `end`: DateTime representing the end date of the range.

 **returns**:
   None

## Return Type
`Future<void>`

## Parameters

- `start`: `DateTime`
- `end`: `DateTime`
