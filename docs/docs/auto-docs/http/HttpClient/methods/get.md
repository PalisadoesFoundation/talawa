# Method: `get`

## Description

Opens an HTTP connection using the GET method.

 The server is specified using [host] and [port], and the path
 (including a possible query) is specified using
 [path].

 See [open] for details.

## Return Type
`Future&lt;HttpClientRequest&gt;`

## Parameters

- `host`: `String`
- `port`: `int`
- `path`: `String`
