# Method: `initialize`

## Description

Initializes the view model with the given event and volunteer group.

 **params**:
 * `parentEvent`: The event associated with the volunteer group.
 * `group`: The volunteer group to be managed.

 **returns**:
   None

## Return Type
`Future<void>`

## Parameters

- `parentEvent`: `Event`
- `group`: `EventVolunteerGroup`
