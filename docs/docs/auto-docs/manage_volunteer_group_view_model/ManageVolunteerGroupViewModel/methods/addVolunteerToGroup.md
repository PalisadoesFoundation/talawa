# Method: `addVolunteerToGroup`

## Description

Adds a volunteer to the specified group.

 **params**:
 * `volunteerId`: The ID of the volunteer to add.
 * `eventId`: The ID of the event.
 * `groupId`: The ID of the group.

 **returns**:
   None

## Return Type
`Future&lt;void&gt;`

## Parameters

- `volunteerId`: `String`
- `eventId`: `String`
- `groupId`: `String`
