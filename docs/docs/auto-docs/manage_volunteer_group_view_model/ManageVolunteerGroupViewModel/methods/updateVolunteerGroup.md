# Method: `updateVolunteerGroup`

## Description

Updates the details of a volunteer group.

 **params**:
 * `group`: The volunteer group to update.
 * `eventId`: The ID of the event.
 * `name`: The new name for the group.
 * `volunteersRequired`: The new number of volunteers required.

 **returns**:
   None

## Return Type
`Future&lt;void&gt;`

## Parameters

- `group`: `EventVolunteerGroup`
- `eventId`: `String`
- `name`: `String`
- `volunteersRequired`: `int`
