# Overview for `VerboseStdio`

## Description

A logger that will print out trace messages.

## Dependencies

- Stdio

## Members

- **stdout**: `io.Stdout`
- **stderr**: `io.Stdout`
- **stdin**: `io.Stdin`
- **filter**: `String Function(String)?`
  If provided, all messages will be passed through this function before being logged.
