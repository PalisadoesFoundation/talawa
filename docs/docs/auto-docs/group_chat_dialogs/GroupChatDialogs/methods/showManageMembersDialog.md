# Method: `showManageMembersDialog`

## Description

Shows manage members dialog (admin only).

 **params**:
 * `context`: The build context
 * `model`: The GroupChatViewModel instance
 * `chatId`: The ID of the chat
 * `chat`: The chat object
 * `onMemberRemoved`: Callback for when a member is removed

 **returns**:
   None

## Return Type
`void`

## Parameters

- `context`: `BuildContext`
- `model`: `GroupChatViewModel`
- `chatId`: `String`
- `chat`: `Chat`
- `onMemberRemoved`: `VoidCallback`
