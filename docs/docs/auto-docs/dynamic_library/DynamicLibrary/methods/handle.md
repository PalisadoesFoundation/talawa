# Method: `handle`

## Description

The opaque handle to the dynamic library.

 Similar to the return value of
 [dlopen(3)](https://man7.org/linux/man-pages/man3/dlopen.3.html).
 Can be used as arguments to other functions in the `dlopen` API
 through FFI calls.

## Return Type
`Pointer&lt;Void&gt;`
