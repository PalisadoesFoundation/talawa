# Method: `_formatRecurrenceRule`

## Description

Formats recurrence rule for display.

 **params**:
 * `context`: BuildContext for localization.
 * `rule`: RecurrenceRule object to format.

 **returns**:
 * `String`: Formatted recurrence rule string.

## Return Type
`String`

## Parameters

- `context`: `BuildContext`
- `rule`: `RecurrenceRule`
