# Method: `formatShortDate`

## Description

Formats the date using a short-width format.

 Includes the abbreviation of the month, the day and year.

 Examples:

 - US English: Feb 21, 2019
 - Russian: 21 февр. 2019 г.

## Return Type
`String`

## Parameters

- `date`: `T`
- `localizations`: `MaterialLocalizations`
