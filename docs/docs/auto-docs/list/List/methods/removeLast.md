# Method: `removeLast`

## Description

Removes and returns the last object in this list.

 The list must be growable and non-empty.
 ```dart
 final parts = <String>['head', 'shoulder', 'knees', 'toes'];
 final retVal = parts.; // toes
 print(parts); // [head, shoulder, knees]
 ```

## Return Type
`E`
