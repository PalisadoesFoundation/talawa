# Method: `+`

## Description

Add [other] to `this`, as if both points were vectors.

 Returns the resulting "vector" as a Point.

 Example:
 ```dart
 var point = const Point(10, 100) + const Point(10, 10); // Point(20, 110)
 point = const Point(-10, -20) + const Point(10, 100); // Point(0, 80)
 ```

## Return Type
`Point&lt;T&gt;`

## Parameters

- `other`: `Point&lt;T&gt;`
