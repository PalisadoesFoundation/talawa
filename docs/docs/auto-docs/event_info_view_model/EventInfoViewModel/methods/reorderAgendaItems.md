# Method: `reorderAgendaItems`

## Description

Method to redorder the sequence of agenda items.

 **params**:
 * `oldIndex`: old index of the item
 * `newIndex`: new index of the item

 **returns**:
   None

## Return Type
`Future&lt;void&gt;`

## Parameters

- `oldIndex`: `int`
- `newIndex`: `int`
