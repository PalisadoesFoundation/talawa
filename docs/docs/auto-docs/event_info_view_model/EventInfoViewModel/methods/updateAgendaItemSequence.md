# Method: `updateAgendaItemSequence`

## Description

Method to update the sequence of an agenda item.

 **params**:
 * `itemId`: id of the agenda item whose sequence need to be updated
 * `newSequence`: new sequence of the item

 **returns**:
   None

## Return Type
`Future&lt;void&gt;`

## Parameters

- `itemId`: `String`
- `newSequence`: `int`
