<div class="section desc markdown">

Controls whether we should try to imply the leading widget if null.

If true and `AppBar.leading` is null, automatically try to deduce what
the leading widget should be. If false and `AppBar.leading` is null,
leading space is given to `AppBar.title`. If leading widget is not null,
this parameter has no effect.

</div>
