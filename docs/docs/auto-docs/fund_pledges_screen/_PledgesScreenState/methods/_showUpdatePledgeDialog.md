# Method: `_showUpdatePledgeDialog`

## Description

Shows the dialog to update an existing pledge.

 **params**:
 * `context`: the build context.
 * `model`: the data model to add pledge data.
 * `pledge`: the pledge being updated.

 **returns**:
   None

## Return Type
`void`

## Parameters

- `context`: `BuildContext`
- `model`: `FundViewModel`
- `pledge`: `Pledge`
