# Method: `fromBasicJsonToList`

## Description

Converts a JSON array to a list of OrgInfo instances.

 **params**:
 * `data`: The JSON array to be parsed.

 **returns**:
 * `List&lt;OrgInfo&gt;`: A list of OrgInfo objects containing the parsed data.

## Return Type
`List&lt;OrgInfo&gt;`

## Parameters

- `data`: `Map&lt;String, dynamic&gt;?`
