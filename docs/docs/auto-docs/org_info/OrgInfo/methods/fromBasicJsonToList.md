# Method: `fromBasicJsonToList`

## Description

Converts a JSON array to a list of OrgInfo instances.

 **params**:
 * `data`: The JSON array to be parsed.

 **returns**:
 * `List<OrgInfo>`: A list of OrgInfo objects containing the parsed data.

## Return Type
`List<OrgInfo>`

## Parameters

- `data`: `Map<String, dynamic>?`
