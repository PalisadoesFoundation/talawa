# Method: `fromJsonToList`

## Description

Converts a JSON array to a list of OrgInfo instances.

 **params**:
 * `json`: The JSON array to be parsed.

 **returns**:
 * `List<OrgInfo>`: A list of OrgInfo objects containing the parsed data.

## Return Type
`List<OrgInfo>`

## Parameters

- `json`: `dynamic`
