# Method: `getPledgesByCampaign`

## Description

This function is used to fetch pledges by campaign ID.

 **params**:
 * `campaignId`: ID of the campaign to fetch pledges for.

 **returns**:
 * `Future&lt;List&lt;Pledge&gt;&gt;`: List of pledges.

## Return Type
`Future&lt;List&lt;Pledge&gt;&gt;`

## Parameters

- `campaignId`: `String`
