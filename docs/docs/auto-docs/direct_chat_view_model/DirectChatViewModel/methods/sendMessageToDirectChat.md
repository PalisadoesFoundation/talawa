# Method: `sendMessageToDirectChat`

## Description

This function send the message to Direct Chat.

 **params**:
 * `chatId`: id of a chat where message need to be send.
 * `messageContent`: content of a message.

 **returns**:
   None

## Return Type
`Future<void>`

## Parameters

- `chatId`: `String`
- `messageContent`: `String`
