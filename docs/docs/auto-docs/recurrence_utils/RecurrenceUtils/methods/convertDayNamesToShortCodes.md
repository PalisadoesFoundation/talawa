# Method: `convertDayNamesToShortCodes`

## Description

Convert day names to short codes (MO, TU, etc.) for API.

 **params**:
 * `days`: Set of day names from WeekDays class

 **returns**:
 * `List<String>`: List of day short codes

## Return Type
`List<String>`

## Parameters

- `days`: `Set<String>`
