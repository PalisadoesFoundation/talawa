# Method: `convertShortCodesToWeekDays`

## Description

Convert short day codes (MO, TU, etc.) to weekday names.

 **params**:
 * `dayCodes`: List of day short codes

 **returns**:
 * `Set&lt;String&gt;`: Set of weekday names

## Return Type
`Set&lt;String&gt;`

## Parameters

- `dayCodes`: `List&lt;String&gt;`
