# <span class="kind-class">MemberNameTile</span> class

</div>

<div class="section desc markdown">

This widget returns a tile containing the name of the member.

Along with the name, there is a circle avatar which either contains the
image uploaded by the user or the first character of his/her name in
uppercase.

</div>
