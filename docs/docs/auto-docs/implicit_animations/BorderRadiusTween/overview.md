# Overview for `BorderRadiusTween`

## Description

An interpolation between two [BorderRadius]s.

 This class specializes the interpolation of [Tween&lt;BorderRadius&gt;] to use
 [BorderRadius.lerp].

 See [Tween] for a discussion on how to use interpolation objects.

## Dependencies

- Tween

## Constructors

### Unnamed Constructor
Creates a [BorderRadius] tween.

 The [begin] and [end] properties may be null; the null value
 is treated as a right angle (no radius).
