Robot Patternâ€‹

We follow the Robot Pattern to keep our test code clean, maintainable, and readable. Each robot encapsulates the logic for interacting with a specific screen or feature. For example:

final loginFormRobot = LoginFormRobot(tester);
await loginFormRobot.loginAdmin();

All robot files are located in:

integration_test/robots/
